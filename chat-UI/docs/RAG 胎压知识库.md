标题：胎压范围说明
      - 胎压正常范围为2.6-3.4bar，随季节温度变化可能引起胎压浮动。
      - 低于2.6bar时，建议补气至2.9bar，并检查是否漏气。


一级提醒

二级提醒

胎压范围
- 报警阈值：2.28bar
- 解除阈值：2.48bar
- 报警阈值：1.8bar
- 解除阈值：2.0bar
是否可驾驶
- 可继续行驶
- 不可行驶，会损坏轮胎
出现场景
- 缓慢漏气未处理到2.3
- 轮胎扎钉，慢速漏气到2.3
- 轮胎扎丁，快速漏气到2.3，并迅速漏气到1.8触发二级报警
- 快速降低到1.8以下

用户行为
- 检查轮胎是否扎钉，自行补气
- 补气后仍然漏气，自驾到店或上门补胎
- 预约上门补胎
引导方案
- 行车：谨慎驾驶，检查轮胎状态并尝试补气
- 驻车：检查轮胎状态，并尽快补气到2.9

- 行车：靠边停车，联系客服获得支持
- 驻车：勿驾驶车辆，并预约上门服务
